# README

## How to build and run code

```bash
$ zig build-exe <file_name>.zig
$ ./<file_name>
```
## How to build object

```bash
$ zig build-obj <file_name>.zig
```

## How to test

```bash
$ zig test <test_file_name>.zig
```

## How to generate documentation

```bash
$ zig test -femit-docs <file_name>.zig
```
